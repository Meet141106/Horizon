# Project Polis — PS-03

## What it does
Dual-mode civic issue reporting app with a 3D cyberpunk city map 
and a synced Kanban board.

## Features
- Double-click city map to drop issue pins with zone auto-detection
- 7 named city zones (Neon Market, Civic Core, Tech Quarter, 
  Skybridge West, Residential Ring, Industrial Belt, Harbor Fringe)
- Kanban board (New / In Progress / Resolved) synced to map in real-time
- Sphere glow changes colour on status update
- Community upvoting with live re-sort
- Scrollytelling overlay on map exploration
- Particle FX on issue resolution
- Fully responsive (mobile tabbed, tablet 2+1, desktop 3-col)
- localStorage persistence

## Tech Stack
- Three.js + GLTFLoader + CSS2DRenderer (3D map)
- Vanilla JS (state management)
- Vite (build)

## Run locally
npm install
npm run dev
